# Cloud Smart Gallery

A React and Vite client for uploading images to a cloud-backed gallery and browsing AI-generated tags.

## Scripts

- `npm run dev` starts the local Vite server.
- `npm run build` creates a production build.
- `npm run lint` checks the client source with ESLint.

## Environment

Set `VITE_API_URL` to the backend API URL. When it is not provided, the app uses `http://localhost:5000` for local development.
